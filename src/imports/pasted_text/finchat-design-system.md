# FinChat — Figma Make 디자인 프롬프트

---

## 서비스 개요

서비스명: FinChat
슬로건: "내 소비를 대화로 이해하다"
플랫폼: 웹 서비스 (SPA, 반응형 웹)
       — 네이티브 앱이 아닌 브라우저 기반 웹 서비스
       — PC·태블릿·모바일 브라우저 모두 대응
콘셉트: 사용자의 카드·계좌 소비 데이터를 바탕으로
       AI가 소비 성향을 분석하고,
       챗봇 형태로 소비 이유·패턴·절약 포인트를
       쉽고 자연스럽게 설명해주는 금융 웹 서비스
타겟: 소비관리가 어려운 2030 대학생, 사회초년생, 직장인
핵심 가치:
       - 복잡한 금융 데이터를 쉬운 말로 설명
       - 단순 조회가 아니라 해석과 상담 제공
       - 내 소비를 "리포트"가 아니라 "대화"로 이해하게 만들기

## 페이지 흐름 요약 (Flow)

랜딩 (0-1)
 ├─ 무료로 시작하기 → 회원가입/로그인 (0-2)
 │                     → 금융 데이터 연동 (0-3)
 │                     → 홈 대시보드 (1)
 └─ 로그인 → 홈 대시보드 (1)

홈 (1)
 ├─ 최근 거래 → 소비 내역 조회 (2)
 ├─ 분석 카드 → 소비 분석 리포트 (3)
 ├─ 챗봇 바로가기 → AI 금융 챗봇 (4)
 ├─ 리포트 카드 → 월간/주간 리포트 (5)
 ├─ 소비 성향 카드 → 소비 성향 프로필 (6)
 ├─ 예산 카드 → 목표 예산 설정 (7)
 ├─ 알림 카드 → 소비 알림 / 위험 신호 (8)
 └─ 프로필 → 마이페이지 / 연동관리 (9)

## 디자인 요청 사항

- 전체적으로 Toss 스타일 금융 웹 서비스처럼 보여야 함
- 카드 중심 레이아웃, 여백 넉넉하게
- 너무 복잡한 표보다 한눈에 보이는 숫자와 짧은 요약 중심
- 챗봇 화면은 친근하지만 금융 서비스답게 신뢰감 있어야 함
- 차트는 과하지 않고 미니멀하게
- 반응형 웹으로 데스크톱 / 태블릿 / 모바일 시안 모두 고려
- 웹 서비스 MVP 수준으로 실제 구현 가능한 구조로 설계
- 피그마 메이크에서 바로 화면 설계 가능한 수준으로 페이지별 UI를 구체적으로 표현
